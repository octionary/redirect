# redirect

Persistent redirect URLs for QR codes and print materials, hosted on GitHub Pages.

## How it works

Each directory contains an `index.html` that redirects to a target URL via `<meta http-equiv="refresh">` and `window.location.href`. To change where a link points, edit the URL in that directory's `index.html`.
